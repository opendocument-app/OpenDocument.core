# ODF implementation

## General

this implementation relies on [ZIP](../zip/README.md) and [SVM](../svm/README.md)

## Features

### Functional

- [x] open
    - [x] decryption
- [x] meta data
- [x] edit
    - [x] text
- [x] save
    - [ ] encryption

### Styles

- [x] font
    - [x] size
    - [x] italic, bold
    - [x] alignment
    - [x] underline, strike through
    - [x] color
    - [x] family
    - [ ] superscript, subscript
- [x] links
- [x] tables
- [x] images
    - [x] svm
- [x] listings
    - [x] bullets
    - [ ] numbering
- [x] drawings
    - [x] line, rect, circle
    - [ ] custom shapes support odf custom-shape #159
    - [ ] transform (e.g. flip, rotate)
- [x] odp slide master / odg page master
- [ ] annotations
- [x] paging (e.g. style slides, center)
